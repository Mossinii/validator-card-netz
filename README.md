# validator-card-netz

Projeto técnico NETZ para validação de cartões de crédito.

Este repositório implementa a validação de números de cartão usando o **Algoritmo de Luhn** e identifica as bandeiras **Visa**, **Mastercard** e **American Express**.

Desenvolvido com o suporte de **GitHub Copilot**, o código segue uma abordagem enxuta e voltada para confiabilidade em cenários de validação de pagamentos.

## Recursos principais

- Validação de integridade do número via Luhn
- Detecção de bandeira (Visa / Mastercard / Amex)
- Código simples e projetado para integração em pipelines de verificação
